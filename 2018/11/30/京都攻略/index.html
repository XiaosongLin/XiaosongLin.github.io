京都攻略

住宿攻略

首选京都站/然后是四条河原町，这两个地方附近。

路线推荐

D1 清水坂商业街(10分钟) → 清水寺(2小时) -> 三年坂二年坂(1.5小时) → 衹园(2小时) → 鸭川（散步吃饭）(1小时) → 八坂神社(1小时)

D2 岚山 → 四条河原町（逛街购物）(2小时)

D3 金阁寺(1.5小时) → 京都二条城(1.5小时) → 京都锦市场(1.5小时) → 先斗町(1小时) → 伏见稻荷大社(2小时) → 四条河原町(2小时)

推荐500日元的巴士1日券，现场购买。
京都巴士也可以使用ICOCA一卡通，该卡适用于关西大部分交通方式，但无优惠力度，可考虑1日券+ICOCA的组合形式

景点推荐

京都景点太多太杂，必去的有这几个，其余随意安排

  • 伏见稻荷大社
  • 清水寺
  • 金阁寺

  • 京都二条城

  • 衹园 （属于逛街的）

  • 鸭川 （属于逛街的）
  • 岚山 （这个地方是一个区域，需要大半天时间，有空可单独安排）

接下去景点按推荐路线来，可删除几个压缩成两天。

Day1

第一天的游览方式以步行为主，这条线路是京都最精华的景点，从清水寺开始，途径三年坂二年坂，再到祇园，最后以八坂神社的夜景结束。日式风格的小街巷两旁，贩卖着各种传统工艺品和小吃，穿着和服的少女最适合边走边逛。

清水坂商业街

肯定会走到的地方，但不推荐在这里买东西
清水坂是清水寺门前一条石板街，这条古朴的街道是参拜清水寺的主要通道。
街道两旁是各种京都特产和工艺品商店，清水烧陶瓷、八桥饼、京果子、京茶果、京扇等都可以在这里找到。
清水坂在七味家本铺处延伸出产宁坂和五条坂两条小路，五条坂上以卖清水烧的商铺和艺廊居多，产宁坂则是从清水寺到高台寺的捷径

清水寺门前街道，由清水寺与高台寺景区步行均可到达

清水寺 10:00 AM

必打卡，很多妹子是在这里租合服

  • 清水寺是京都最古老的寺院，与金阁寺、二条城并称为京都三大名胜，1994年列入世界文化遗产名录。
  • 进入清水寺内，会经过仁王门、西门、三重塔、随求堂、开山堂等建筑，还能看到仁王像、地藏佛等雕刻。
  • 其中清水舞台是日本国宝级文物，春季时樱花烂漫，是京都的赏樱名所之一，秋季时红枫飒爽，又是赏枫胜地。
  • 清水舞台下的音羽瀑布，其三个源流分别代表着健康、学业和姻缘，很多人在这里排队接水饮用、祈福。

Tips：

  • 清水寺的屋顶与正殿目前正在修缮中，预计2020年3月完工。整修中的清水寺本堂仍然可以进入，清水舞台会开放参拜通道，但是有布幕遮挡，会影响拍照效果。
  • 音羽瀑布只能接其中一道水来喝，因为太贪心愿望就不会实现。
  • 寺内有很多游客求签，100日元一次，是好签就好好收着，如果是坏签记得要把它绑在旁边的架子上。

去清水寺步行时候，与清水坂有一条平行的街，千万别走错了，那条路看上去能到但是旁边是一个非常大的墓地, 去年把我们吓了个半死

交通：
①搭乘京阪巴士「83、85、87、88、206路、100路 」在「五条坂」下车，步行10分钟。
②搭乘京阪巴士「207路 」在「清水道」下车，步行10分钟。

到了京都不看清水寺说不过去，但是看了觉得也不过如此，虽然是装修，但是还是值得来的

三年坂二年坂 11：AM

必打卡，因为都来了清水寺

从清水寺折返往山下走，便可以到达最具京都风情的街道三年坂二年坂，

  • 清水寺外面的三年坂二年坂是极具京都风情的坡道，也是日本重要传统的建造物群保护地区。
  • 二年坂从北面的高台寺往南延伸至三年坂，一路连绵着保留了红壳格子和虫笼窗式的古老町家建筑，极具韵味。
  • 二年坂有各种日式小吃店、和风杂货店、宫崎骏动画周边的“橡子共和国”，三年坂有玩偶店、和服店和清水烧瓷器店，是京都买手信的好地方。
  • 传说如果在三年坂不慎跌倒，三年之内必有大难临头，又说若找到卖葫芦的店便可破除魔咒，于是至今都还有一家卖葫芦的“瓢箪屋”坐落在山脚。
  • 可以尝到正宗的宇治抹茶甜品

午餐推荐

三年坂二年坂的菊乃井 米其林三星怀石料理

或者 如果对怀石料理不感兴趣，也可以尝试京都的另一大特色美食汤豆腐，奥丹是一家本地人认为能吃到最传统汤豆腐的店，黄豆是从”滋贺县”的农家所提供；

衹园

走三年坂二年坂完顺路也会到的，花街，都是艺伎，

  • 衹园是指从八坂神社、鸭川到东大路通之间的区域，是京都传统的繁华街区。
  • 这里出名的原因是有舞伎和艺伎出没，其中衹园甲部为京都规模最大的花街，有时可以在花见小路上看到她们的身影。
  • 衹园一带有很多茶屋和料亭，传统建筑和有趣的小店十分值得驻足，是夜生活的好去处。

乘坐「京阪本線」在「祇園四条駅」下车，乘坐市巴士在「祇園站」下车。 或者从三年坂二年坂步行到花见小路约1公里

花见小路是日本名气最大的花街，最精华的部分在四条通南面的一段，可以租一套和服，穿上木屐，一路漫步拍照。路两旁满是有艺伎表演的茶屋和高级料理亭，随便走进旁边的弄巷，更能在深锁的木造门前感受到独特神秘的气氛。或许，有机会见到艺妓画着精致妆容、抱着琴盒、踩着小碎步，在茶屋间穿梭赶场。

鸭川

必打卡，晚上吃饭散步一般跑这里

  • 鸭川是京都的生命河，长31公里，流经京都中央，河水十分清澈。
  • 鸭川沿岸遍布餐厅，可以品尝到京都料理、韩国料理等多国美食。
  • 河流两旁有步行径及单车径

搭乘「京阪本線」在「出町柳駅」、「三条駅」、「祇園四条駅」、「神宮丸太町駅」、「清水五条駅」、「東福寺駅」下车。

鸭川这条线很长，…拍照很棒，不过即便走什么线最终都会碰到鸭川

八坂神社

可选，但是也是连接三年坂二年坂地方，夜间6点亮灯挺好看的，我觉得下午去清水寺刚好可以傍晚吃完饭到这里，或者直接在鸭川去四条河原町逛街

  • 位于京都市东山区的神社，是关西地区最知名且历史最悠久的神社之一。
  • 神社大殿被称为“衹园造”，是日本独特的神社建筑，而舞殿上无数的提灯，让夜晚甚是美丽。
  • 这里有专门求签、写绘马和请御守的地方，绘马图案非常多样，祈求恋爱的心形绘马十分受欢迎。

搭乘「京阪线」在「祇园四条」下车，再沿四条通向东步行5分钟

DAY2 坐小火车游览岚山

第二天的游览方式以火车为主，先坐JR从京都站到岚山，再换乘岚山特色小火车游览，下午可以逛一逛周边景点

岚山是一片区域的统称，岚山区域有诸多景点，比较知名的有岚山竹林，天龙寺和渡月桥。

绝对是很舒服很舒服的地方

  • 岚山地区由岚山和嵯峨野两块区域组成，这片区域里不仅有丰富的人文景点，还有随四季变换的自然风光与复古小火车。
  • 这里集中了许多古刹和神社，天龙寺更是世界遗产级别的古迹，由野宫神社向大河内山庄之间是著名的竹林小径。
  • 作为赏樱赏枫的圣地，岚山每到春秋两季吸引了大量游人，嵯峨野小火车往往一票难求，而冬夏时节则十分静谧，适合喜欢清幽的旅行者。
  • 岚山地区有不少餐饮老铺、和果子屋以及可爱的杂货店，还可以租赁和服，是个非常适合放慢节奏来发现的地方。

从市区前往岚山一般要搭乘前往岚山方向的岚电「嵐電（京福）嵐山本線」，在岚山站「嵐山駅」下车。
搭乘「山陰本線」在「嵯峨嵐山駅」下车。
搭乘「阪急嵐山線」在「嵐山駅」下车。

小火车票价：
单程，大人620日元，儿童310日元
游船票价：
大人4100日元，小孩2700日元

09:30 岚山小火车

岚山小火车原名嵯峨野游览小火车，全程7.3公里，单程需时约25分钟，由嵯峨站起，经岚山站，保津峡站，至龟冈站止后，原车返回。全车共五节车厢，五号车厢为全开放式，是否销售与当天天气有关。
小火车沿线春天是满山遍野的樱花，夏天是鲜嫩欲滴的新绿，秋天是一片萧红，冬天是则是银装素裹的白，无论何时，乘小火车都是不错的体验。

10:30 保津川游船

保津峡因溪谷之美而闻名，所以，游船是体验保津峡灵动的最佳方式。小木船差不多承载20人，船家会一边撑船一边哼着民谣，歌声在峡谷间飘荡，如果恰逢枫叶季，岸两旁漫山遍野棕红，如同油画一般。走完全程约一个半小时，行驶十分平缓，没有急弯险滩，也不会湿身。

到达方式：乘小火车到达终点站龟冈站后，搭乘摆渡巴士约10分钟前往游船起点。
搭乘指南：船票背面有号码牌，等轮到乘船会广播号码。

午餐推荐

嵯峨岚山站附近有很多网红餐厅，推荐广川鳗鱼饭，这是一家专门做鳗鱼料理的米其林1星餐厅，需要排队，或提前网上预订。

Arabica
在离小火车岚山站附近的渡月桥边上，有一家红遍世界的咖啡店Arabica，被评为京都最美的咖啡馆，拥有270度玻璃视觉，被山水环绕，十分优美。

15:30 嵯峨野竹林

小火车岚山站的附近，有一片100米左右的竹林隧道，这里是电影《卧虎藏龙》的拍摄地，

到达方式：游船上岸后经过渡月桥到对岸，步行1公里。

岚山景点太多，推荐结束后基本傍晚可以去四条河原町这个京都最大的逛街的地方

DAY3 乘巴士逛京都

第三天的游览方式以巴士为主，巴士是京都最方便的公共交通。上午先参观金阁寺，这里是动画片《一休哥》中的寺庙原型；接着拜访德川家康将军的府邸二条城；然后去号称“京都厨房”的锦市场逛吃逛吃；下午游览伏见稻荷大社。

这三个地方都是必打卡，中间穿插的可选地方

金阁寺

必必必必定打卡，虽然走一圈就出来了，不过是国宝级别的

  • 建于1379年，原为足利义满将军（动画聪明的一休中利将军原型）的山庄，后改为禅寺“菩提所”。
  • 这座高调华丽的禅寺，与前方的镜湖池交相辉映，成为京都最经典的画面之一。
  • 金阁寺是3层的楼阁，第一层为法水院；第二层为潮音洞，供奉着观音；第三层是正方形的佛堂，供奉着三尊弥陀佛。
  • 游客拿到的门票是写有祝福话语的朱印，院中的不动堂旁边还有中文和韩文的神签可供占卜。

交通
搭乘巴士「市営204、市営205、市営59、市営101、市営12」在「金閣寺道」下车即可。

开放时间
09:00-17:00

10:00 二条城

必打卡

  • 二条城由德川家康于1603年所建，是德川幕府的权力象征，这里保存了日本桃山时代的绘画雕刻及建筑特色，是京都的世界文化遗产之一。
  • 整座二条城由石墙及护城河包围着，总体上分为本丸御殿、二之丸御殿以及环绕两个御殿的庭园等区域。
  • 亮点是二之丸御殿内的鹂鸣地板，走在上面地板会发出夜莺啼叫般的声响，如此设计是以防夜间刺客入侵。
    -这里也是京都首屈一指的赏樱胜地，可观赏到有“樱花王者”称誉的八重樱。

到达方式：从金阁寺搭乘巴士101、12、204路，车程30分钟。
门票：600日元
开放时间：8:45 -16:00

推荐午餐 京都锦市场

小型心斋桥，时间不够可忽略，吃的蛮多

  • 京都百年市场，旅游购物美食必打卡街，被称为“京都人的厨房”。
  • 锦市场约有140多家店铺，贩卖生鲜食品、各式蔬菜、干货、腌菜和极具京都特色的调料。
    虽然有不少鱼摊小贩，但整体环境非常的干净整洁，喜欢逛吃的人在这里至少得花上两个小时去体验。
    · 大部分店家的营业时间为12-17点，虽然商品不是最便宜的，但这里却是最亲近京都人生活的地方。

Tips:
·各店铺营业时间不同，周三休息的店铺比较多，尽量避开周三前往。

乘坐市营巴士5在「四条高仓（大丸百货店前）」站下车步行2分钟；乘坐地铁乌丸线在「四条」站下车步行 3分中；乘坐地铁东西线在「京都市役所前」站下车步行10分钟。

从二条城做巴士101、12路，车程25分钟。

推荐午餐 先斗町

挺不错的 锦市场/先斗町二选一
如果是京都的锦市场是菜市场一样的话，那么先斗町就是米其林美食街…..怀石料理………….

先斗町是京都最为热闹的一条街道，与鸭川河道平行，从河源町延伸到三条。先斗町大大小小的居酒屋、酒吧、餐馆林立，却十分巧妙的保持了一种统一的古典建筑风格，是京都最有风情的老街之一。

  • 她最有特色的地方在于东侧许多餐馆临水，夏天在鸭川上可以架起纳凉床，经营川床料理，也是来京都的独特体验。
    道路两侧的居民楼上挂满了大大小小的酒吧招牌，忍不住想去喝一杯，除了正常的酒吧以外这里还有专门为女性设置的女性酒吧。。

搭乘京阪电车，三条站下车，步行5分钟。

14:00 伏见稻荷大社

必打卡

日本约有1/3的神社属于稻荷神社，伏见稻荷大社是稻荷神社的总本社，地位崇高、历史悠久。
伏见稻荷大社供奉着保佑商业繁荣、财运亨通、五谷昌盛的稻荷大神，香客众多。
入口处矗立着由丰臣秀吉于1589年捐赠的大鸟居，后面是神社的主殿及其他建筑物，尤以“千本鸟居”最为著名。
密集的朱红色“千本鸟居”，是京都代表性景观之一，也曾出现在电影《艺伎回忆录》中。
狐狸被视为神明稻荷的使者，因此在神社里你会看到各式的狐狸石像、狐狸脸形的绘马（祈愿牌）、狐狸造型的御守（护身符），非常可爱。
鸟居通道里也有一些餐厅提供当地的菜肴，如稻荷寿司和狐族乌冬面，汤面上放有炸油豆腐，据说是狐狸最喜欢的食物。

是个人都会在这里拍照… 请使用长焦虚化掉背景的人类… 网红拍照圣地

时间:
8:30～16:30

交通方式:
• 京阪电车伏见稻荷车站下车，步行约5分钟
• JR奈良线稻荷车站下车，步行约3分钟

从锦市场出发，坐京阪线至伏见稻荷站，车程20分钟。


购物推荐

虽然京都和大阪、东京相比，大型百货数量不多，但是京都的商店以高品质和良好的服务态度而闻名。对于追求购物品位和质量，喜欢悠闲shopping的购物狂来说，京都十分理想。大部分商场主要集中在京都市中心河源町和京都JR站附近，不会过于奔波。
